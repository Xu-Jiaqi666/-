# 动态投资组合优化系统（省级大创项目）

**负责人**：徐嘉奇（湘潭大学 数据科学与大数据技术 2023级）  
**项目时间**：2025.09 – 2026.04
**指导老师**：杨柳教授、张建平教授  

## 📌 项目简介
面向A股动态投资组合优化问题，构建“情景感知→收益预测→分布鲁棒优化→动态调仓”框架，融合多源协变量（个股收益、沪深300、汇率、国债收益率、政策指数等）。

## 🔧 核心方法
- 分布鲁棒优化（DRO）+ CVaR尾部风险约束
- 对偶SDP转化 + CVXPY/SCS滚动优化求解
- SHAP模型可解释性分析

## 📊 主要结果
- 基准策略年化收益率：**17.55%**
- 夏普比率：**0.677**（优于均权组合）

## 📁 文件说明
- `code/` : 核心优化与回测代码（Jupyter Notebook）
- `data/` : 原始数据文件（股票、指数、宏观变量）
- `results/` : 回测结果与SHAP分析图
